오늘 새로 알게 된 용어를 **이해·암기용**으로 정리한다. "정의 + 비유 + 예시"로 짧게, 주제별로 묶었다.

## 🌐 네트워크

**IPv6** — 128비트로 확장된 인터넷 주소 체계. IPv4(32비트, 약 43억 개)가 고갈돼 나왔다.
비유: 자릿수가 부족해진 옛 우편번호를 대폭 늘린 새 체계. 예: `2001:db8::1` (총 약 3.4×10³⁸개).

**TTL(Time To Live)** — 이름은 같지만 두 뜻이다. ① **IP 패킷 TTL** = 패킷이 지날 수 있는 라우터(홉) 수 제한, 0이 되면 폐기(무한 루프 방지). ② **DNS·캐시 TTL** = 데이터를 몇 초 캐시할지.
비유: ① 택배가 거칠 수 있는 경유지 횟수 ② 우유 유통기한. 예: `traceroute`가 TTL을 1씩 늘려 경로를 추적.

**OSI 7계층** — 네트워크 통신을 7단계(물리→데이터링크→네트워크→전송→세션→표현→응용)로 나눈 표준 모델.
비유: 편지가 작성→우체통→분류→운송→배달로 단계별 처리되는 것. 예: 문제 생기면 "몇 계층 문제인가"로 나눠 진단.

**DDNS(동적 DNS)** — 통신사가 수시로 바꾸는 유동 IP를 고정된 도메인 이름에 자동으로 따라붙게 해주는 것.
비유: 이사를 다녀도 같은 이름표로 찾아오게 하는 우편 전달 서비스. 예: 집 서버를 외부에 노출할 때 사용.

## 🔐 보안·인증

**환경변수(.env)** — API 키·비밀번호 같은 비밀값을 코드 밖에 두고 실행할 때만 불러 쓰는 것.
비유: 금고 열쇠를 소스코드에 적지 않고 따로 보관. 예: `.env`에 넣고 `.gitignore`로 깃 업로드 차단.

**해시(Hash)** — 입력을 고정 길이 값으로 바꾸는 **단방향** 변환(되돌리기 어려움).
비유: 재료를 갈아 만든 스무디 — 원래 재료로 못 되돌린다. 예: 비밀번호를 원문 대신 해시로 저장.

**bcrypt** — 비밀번호를 안전하게 저장하기 위한 해시 알고리즘. 일부러 느리게 설계해 무차별 대입을 방어.
비유: 일부러 뻑뻑하게 만든 자물쇠. 예: 로그인 비밀번호 저장에 bcrypt 사용.

**쿠키(Cookie)** — 사이트가 준 작은 정보를 **브라우저가 보관**했다가 방문할 때마다 보여주는 것.
비유: 지갑 속 작은 회원카드. 예: 로그인 상태 유지.

**세션(Session)** — 서버가 쿠키 번호를 보고 **금고에서 꺼내는 내 정보**(장바구니·권한 등).
비유: 사이트 금고 속 내 서류철. 예: 쿠키 = 카드 번호, 세션 = 그 번호로 찾는 실제 정보.

**HTTPS / SSL 인증서** — HTTP를 암호로 감싼 안전한 버전.
비유: 엽서(HTTP, 중간에서 보임) vs 밀봉 봉투(HTTPS). 예: 주소창 자물쇠 아이콘, 로그인·결제엔 필수.

## 🌍 웹 통신

**HTTP(HyperText Transfer Protocol)** — 브라우저와 서버가 웹에서 정보를 주고받는 규칙. (어제 정리, 오늘 복습)
비유: "이 페이지 줘"(요청) → "여기 있어"(응답)의 대화 규칙.

**HTTP 상태 코드** — 서버 응답에 붙는 세 자리 숫자. 앞자리만 보면 된다.
- `2xx` 성공(초록불) · `3xx` 이동(노란불) · `4xx` 내(요청) 잘못 · `5xx` 서버 잘못
- 예: `200 OK`, `301/302` 주소 이동, `401` 로그인 필요, `403` 권한 없음, `404` 없는 페이지, `429` 요청 과다, `500` 서버 오류.

## 🛠 개발·배포

**배포(Deploy)** — 만든 서비스를 사용자가 쓸 수 있게 서버/플랫폼에 올려 운영하는 것.
비유: 시제품을 실제 매장에 진열. 예: 화면은 Vercel, 서버는 Render에 올리기.

**커밋(Commit)** — 코드의 "저장 지점" 하나. 변경사항을 기록한다.
비유: 게임의 세이브 포인트. 예: `git commit`.

**브랜치(Branch)** — 코드의 갈래. 메인을 건드리지 않고 따로 작업하는 평행 줄기.
비유: 나무 줄기에서 갈라진 가지. 예: 기능 개발용 `feature` 브랜치.

**GitHub Actions** — GitHub가 제공하는 서버에서 빌드·테스트·배포를 자동 실행하는 기능.
비유: 코드를 올리면 자동으로 돌아가는 컨베이어벨트. 예: 내 블로그도 push하면 Actions가 자동 빌드·배포한다.

**usage(사용량)** — 서비스·API를 얼마나 썼는지(과금·한도 기준).
비유: 전기 계량기. 예: API 호출 횟수, 클라우드 사용량.
